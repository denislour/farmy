# Farm Stack

This is a skeleton project for the FARM stack.

## Start project

- By VS code

  > Go to Run And Debug
  > Run Farmy

- By command line
  > cd backend
  > uvicorn app.api.server:app --reload --lifespan on

## Run test

- By VS code

  > Go to Run And Debug
  > Run Farmy Test

- By command line
  > cd backend
  > pytest -v
